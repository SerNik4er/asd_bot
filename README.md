🤖 asd_bot

Телеграм-бот для помощи и поддержки.

📋 Описание

Этот бот создан для помощи детям с расстройствами аутистического спектра и их родителям**. Бот помогает отслеживать рутину, даёт напоминания и предоставляет полезные материалы.

✨ Основные возможности

- 🔐 Безопасное хранение токена через переменные окружения
- 🗄️ Работа с базой данных SQLite
- 📅 Планировщик задач (напоминания, регулярные рассылки)
- ⌨️ Удобная клавиатура для навигации
- 🧠 Машина состояний для сложных диалогов

🛠️ Технологии

- Python 3.10+
- python-telegram-bot (v20+)
- SQLite3
- python-dotenv

🚀 Установка и запуск

1. Клонирование репозитория

bash
git clone https://github.com/SerNik4er/asd_bot.git
cd asd_bot

2. Установка зависимостей
bash
pip install -r requirements.txt

3. Настройка переменных окружения
Создайте файл .env в корневой папке проекта и добавьте в него ваш токен:

text
BOT_TOKEN=ваш_токен_от_BotFather
4. Запуск бота
bash
python bot.py
