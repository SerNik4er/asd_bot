# 🤖 ASD Bot

[![Telegram](https://img.shields.io/badge/Telegram-@pwk_rasbot-blue?logo=telegram)](https://t.me/pwk_rasbot)

Телеграм-бот для помощи и поддержки.

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)



## 📋 Описание

Этот бот создан для помощи детям с расстройствами аутистического спектра и их родителям. Бот помогает отслеживать срывы, даёт напоминания и предоставляет полезные материалы.

## ✨ Основные возможности

- 🔐 **Безопасность**: токен хранится в переменных окружения, не попадает в код
- 🗄️ **База данных**: работа с SQLite для хранения пользовательских данных
- 📅 **Планировщик**: регулярные напоминания и запланированные задачи
- ⌨️ **Удобная навигация**: интерактивные клавиатуры для всех действий
- 🧠 **Машина состояний**: поддержка сложных многошаговых диалогов

## 🛠️ Технологии

- Python 3.10+
- python-telegram-bot (v20+)
- SQLite3
- python-dotenv
  
> 💡 **Точные версии зависимостей** указаны в файле `requirements.txt`.

## 🚀 Установка и запуск

### 1. Клонирование репозитория

bash
git clone https://github.com/SerNik4er/asd_bot.git
cd asd_bot
2. Установка зависимостей
bash
pip install -r requirements.txt
